Aplicaciones Web Progresivas

¿Qué es una PWA?

Una PWA es una aplicación web que utiliza tecnologías web progresivas para proporcionar una experiencia de usuario similar a la de una aplicación nativa.

Las PWA son aplicaciones web que se comportan como aplicaciones nativas. Pueden funcionar sin conexión a Internet, enviar notificaciones push, acceder a la cámara y micrófono del dispositivo, y mucho más.
